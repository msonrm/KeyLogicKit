# ゲームパッド入力 マッピング仕様書

ゲームパッド（W3C Standard Gamepad Layout 準拠）で日本語・英語・韓国語を入力する方式。
Start ボタンでモードを切り替える。

## 設計原則

1. **左手=子音/行選択、右手=母音/文字選択**: ゲームパッドの左右分離を活用
2. **同時押しで1文字入力**: D-pad（左手）とフェイスボタン（右手）の chord
3. **Eager output + rollback**: ボタン押下の瞬間に即出力し、300ms 以内の変更で差し替え
4. **後置シフト**: 入力後に修飾（拗音、濁点、子音サイクル等）。同時押し数を削減
5. **モード共通操作**: LS 押込み=確定/改行、RS 押込み=キャンセル、右スティック←=BS

## 入力モード

| モード | バッジ色 | 初期 | パイプライン |
|--------|---------|------|-------------|
| 日本語 | 青 | ✓ | GCController → GamepadInputManager → **InputManager**（IME 経由） |
| 韓国語 | 紫 | | GCController → GamepadInputManager → **onDirectInsert**（IME バイパス） |
| 英語 | 緑 | | GCController → GamepadInputManager → **onDirectInsert**（IME バイパス） |

サイクル順（Start ボタン）: **日本語 → 韓国語 → 英語 → 日本語**

---

## 共通操作

| ボタン | 動作 |
|--------|------|
| LS 押し込み | 確定（composing 中、文節単位の部分確定）/ 改行（idle 時） |
| RS 押し込み | キャンセル（composing 破棄） |
| Back / Select | スペース |
| Start | モード切替 |
| 右スティック← | バックスペース（1文字削除） |

### 左アナログスティック

| 状態 | ← | → | ↑ | ↓ |
|------|---|---|---|---|
| idle（全モード） | カーソル左 | カーソル右 | — | — |
| composing（日本語） | 文節縮小 | 文節拡張 | 前候補 | 変換開始 |
| selecting（日本語） | 文節縮小 | 文節拡張 | 前候補 | 次候補 |

英語・韓国語モードでは常にカーソル移動。

## 日本語モード

### 子音行（D-pad + LB）

| 操作 | 行 | 操作 | 行 |
|------|----|------|----|
| なし | あ行 | LB | は行 |
| ← | か行 | LB+← | ま行 |
| ↑ | さ行 | LB+↑ | や行 |
| → | た行 | LB+→ | ら行 |
| ↓ | な行 | LB+↓ | わ行 |

### 母音（フェイスボタン + RB）

| ボタン | 母音 |
|--------|------|
| RB | あ (a) |
| X (左) | い (i) |
| Y (上) | う (u) |
| B (右) | え (e) |
| A (下) | お (o) |

### かなテーブル（10行 × 5段）

```
            あ(RB)  い(X)  う(Y)  え(B)  お(A)
なし:        あ      い      う      え      お
←(か行):    か      き      く      け      こ
↑(さ行):    さ      し      す      せ      そ
→(た行):    た      ち      つ      て      と
↓(な行):    な      に      ぬ      ね      の
LB(は行):   は      ひ      ふ      へ      ほ
LB+←(ま行): ま      み      む      め      も
LB+↑(や行): や      「      ゆ      」      よ
LB+→(ら行): ら      り      る      れ      ろ
LB+↓(わ行): わ      ゐ      ？      ゑ      を
```

### 特殊入力

| ボタン | 出力 | 判定 |
|--------|------|------|
| RT 単押し | ん | RT リリース時、LT 関与なし |
| LT + RT 同時 | っ | LT ホールド中に RT → LT リリースで発火 |
| LT 単押し | 拗音後置シフト | 直前文字を拗音/小書きに変換。対象外→っ追加 |

### 右アナログスティック

| 方向 | 動作 |
|------|------|
| ↑ | 濁点/半濁点トグル |
| → | 長音「ー」挿入 |
| ← | バックスペース |
| ↓ 1回 | 「、」挿入 |
| ↓ 2回（400ms以内） | 「、」→「。」に差し替え |

## 英語モード

### T9 ベースレイアウト

ニュートラル（D-pad なし）= T1（記号）、D-pad 時計回り = T2〜T5、LB 系 = T6〜T0。

| 操作 | T9 | RB | X(左) | Y(上) | B(右) | A(下) |
|------|----|----|-------|-------|-------|-------|
| なし | T1 | **1** | **(** | **?** | **)** | **!** |
| ← | T2 | 2 | a | b | c | |
| ↑ | T3 | 3 | d | e | f | |
| → | T4 | 4 | g | h | i | |
| ↓ | T5 | 5 | j | k | l | |
| LB | T6 | 6 | m | n | o | |
| LB+← | T7 | 7 | p | q | r | s |
| LB+↑ | T8 | 8 | t | u | v | |
| LB+→ | T9 | 9 | w | x | y | z |
| LB+↓ | T0 | 0 | @ | # | - | _ |

### RT / 右スティック

| 操作 | 動作 |
|------|------|
| RT 単押し | 数字「0」入力 |
| 右スティック→ | スペース |
| 右スティック→ 2回（400ms以内） | スペース→ピリオドに差し替え |
| 右スティック↓ | カンマ |
| 右スティック↑ | アポストロフィ |
| 右スティック← | バックスペース |

### LT シフト（3段階）

| 操作 | 効果 | 解除条件 | ビジュアライザ表示 |
|------|------|---------|-----------------|
| 短押し | 次の1文字だけ大文字 | 1文字入力後 | SHIFT |
| 2度押し（400ms以内） | スマート Caps Lock | 空白・記号入力で自動解除 | Caps |
| 長押し（0.5秒以上） | 永続 Caps Lock | 再度 LT 長押し | CAPS |

## 韓国語モード

### 設計思想

2ボル式（두벌식）ベース。母音ボタン配置は字形の方向に対応（天地人の方向原理）。

### 子音（D-pad + LB）

D-pad で子音（초성）を選択。ㄱㄴㄷㄹ順（가나다라순）。

| 操作 | 子音 | 操作 | 子音 |
|------|------|------|------|
| なし | ㅇ | LB | ㅁ |
| ← | ㄱ | LB+← | ㅂ |
| ↑ | ㄴ | LB+↑ | ㅅ |
| → | ㄷ | LB+→ | ㅈ |
| ↓ | ㄹ | LB+↓ | ㅎ |

### 母音（フェイスボタン）— 字形方向対応

| ボタン | 位置 | 基本母音 | RT シフト（y系） |
|--------|------|---------|----------------|
| RB | 中立 | ㅡ（水平線） | ㅣ（垂直線） |
| X | 左 | ㅓ（左向き） | ㅕ |
| Y | 上 | ㅗ（上向き） | ㅛ |
| B | 右 | ㅏ（右向き） | ㅑ |
| A | 下 | ㅜ（下向き） | ㅠ |

### 音節入力

子音（D-pad）+ 母音（フェイスボタン）の同時押しで1音節を合成。

Unicode 合成式: `syllable = 0xAC00 + (onset × 21 + nucleus) × 28 + coda`

### 받침（終声）

**LT ホールド + D-pad**（母音ボタンなし）で받침を追加。
LT を先に押してから D-pad で子音を選択（D-pad のエッジで発火）。

### 複合母音・子音修飾

| 操作 | 効果 | 例 |
|------|------|-----|
| RT + フェイスボタン | y系母音 | ㅏ→ㅑ, ㅓ→ㅕ, ㅗ→ㅛ, ㅜ→ㅠ |
| RT 単押しリリース | ㅣ付加 | ㅏ→ㅐ, ㅓ→ㅔ, ㅗ→ㅚ, ㅜ→ㅟ, ㅡ→ㅢ |
| 右スティック↑ | 子音サイクル | ㄱ→ㅋ→ㄲ→ㄱ（平音→激音→濃音） |
| 右スティック→ | 複合母音（ㅏ/ㅓ付加） | ㅗ→ㅘ, ㅜ→ㅝ |
| 右スティック↓ | ピリオド | |

### 入力例: 「한국어」

1. **한**: ㅎ(LB+↓) + ㅏ(B) → 하, LT ホールド + ↑(ㄴ받침) → 한
2. **국**: ㄱ(←) + ㅜ(A) → 구, LT ホールド + ←(ㄱ받침) → 국
3. **어**: ㅇ(なし) + ㅓ(X) → 어

## 入力タイミング

### Eager Output + Rollback

母音ボタン押下の瞬間に即出力し、同時打鍵ウィンドウ（300ms）内に子音が変化した場合は差し替え。全3モード共通のパターン。

```
Frame 0: RB 押下 → 即出力
Frame 3: D-pad← 押下 (50ms後) → 差し替え（300ms以内）
Frame 5: LB 押下 (83ms後) → 再差し替え（300ms以内）
...ボタンリリース → 確定
```

子音ボタンのリリース（ボタン数の減少）では巻き戻しを発火しない。

### 内部パラメータ

| パラメータ | 値 | 用途 |
|-----------|-----|------|
| chordWindow | 300ms | 同時打鍵判定ウィンドウ |
| doubleTapWindow | 400ms | 2度押し判定（句読点、Caps Lock 等） |
| stickThreshold | 0.5 | スティック入力判定閾値 |
| longPressThreshold | 500ms | LT 長押し判定（永続 Caps Lock） |

## ビジュアライザ

### モード別表示

- モードバッジ: ビジュアライザ最上部に常時表示（青/紫/緑）
- D-pad ラベル: モードに応じて子音行名/T9行名/ハングル子音を表示
- フェイスボタン: 日本語=かな、英語=文字（シフト状態反映）、韓国語=母音
- LT/LB 配置: 物理コントローラー準拠（LT=外側、LB=内側）
- RB/RT 配置: RB=内側、RT=外側

### 英語シフト状態

フェイスボタン文字と LT ラベルがシフト状態に連動:
- 通常: 小文字、LT=「SHIFT」
- シフト: 大文字（1文字のみ）、LT=「SHIFT」
- スマート Caps: 大文字、LT=「Caps」
- 永続 Caps: 大文字、LT=「CAPS」

## 実装ファイル

### ネイティブ版（Swift、GIME アプリ）

| ファイル | 役割 |
|----------|------|
| `GamepadResolver.swift` | かなテーブル、英語 T9 テーブル、モード enum、アクション enum |
| `GamepadInputManager.swift` | GCController 接続監視、スナップショット処理、3モード入力ハンドラ |
| `KoreanComposer.swift` | ハングル音節合成エンジン、子音/母音テーブル、サイクル/複合母音マップ |
| `GamepadVisualizerView.swift` | SwiftUI ビジュアライザ（モード別表示、シフト状態反映） |
| `App.swift` | アプリエントリポイント、IMETextView 接続、コールバック配線 |

### Web 版（TypeScript）

| ファイル | 役割 |
|----------|------|
| `web/src/engine/gamepad-kana-table.ts` | かなテーブル、拗音マップ、濁点マップ |
| `web/src/hooks/useGamepadInput.ts` | rAF ポーリング、エッジ検出、eager output + rollback |
| `web/src/components/GamepadVisualizer.tsx` | ビジュアライザ（日本語モードのみ） |

### アーキテクチャ

```
キーボード:    pressesBegan → KeyEvent → KeyRouter → InputManager
ゲームパッド:  GCController → GamepadSnapshot → GamepadInputManager
                → InputManager（日本語: IME 経由）
                → onDirectInsert（英語/韓国語: IME バイパス）
```
