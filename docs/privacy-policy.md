# Privacy Policy — Logical Layout Labo

*Last updated: March 13, 2026*

## Overview

Logical Layout Labo ("the App") is a Japanese keyboard layout experimentation tool for iPad. This privacy policy explains how the App handles user data.

## Data Collection

**The App does not collect, store, or transmit any personal data.**

Specifically:

- **No keystroke logging** — The App intercepts hardware key events solely for real-time Japanese input conversion (IME processing). Keystrokes are consumed immediately for text conversion and are not recorded, stored, or transmitted.
- **No analytics or tracking** — The App contains no analytics SDKs, no tracking pixels, and no telemetry of any kind.
- **No network communication** — The App makes no network requests. All text conversion and processing happens entirely on your device.
- **No user accounts** — The App does not require or support user accounts or sign-in.

## On-Device Storage

The App stores the following data locally on your device only:

- **User preferences** — Keyboard layout selection and display settings, stored via UserDefaults (Apple's standard on-device preferences API).
- **Imported keymap files** — If you import custom keyboard layout definitions (JSON files), they are stored in the App's sandboxed documents directory on your device.

This data never leaves your device and is automatically deleted when you uninstall the App.

## Third-Party Services

The App does not integrate with any third-party services, advertising networks, or analytics platforms.

## Open-Source Components

The App uses the following open-source library for on-device Japanese text conversion:

- **AzooKeyKanaKanjiConverter** (MIT License) — A dictionary-based kana-kanji conversion engine that runs entirely on-device.

## Children's Privacy

The App does not collect any data from any users, including children.
